## Deployment

To deploy this project run

for backend
```bash
cd dwanp-backend
npm install

```
to run the server
```bash
cd dwanp-backend
nodemon Server.js

```
for frontend
```bash
cd dwanp-frontend
npm install

```
to run the react app
```bash
cd dwanp-frontend
npm start

```

## Run Locally

Clone the project

```bash
  git clone https://github.com/NaVindu0315/PUSL3120-FullStack-Group-01.git
```

Go to the project directory

```bash
  cd dwanp-backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  nodemon Server.js
```
